# Installation And Usage:

## To run this project run

-  git clone https://github.com/P666R/Albums-List-React-App.git
  
## Go to directory

-  cd Albums-List-React-App-main
  
## Install all dependencies

- npm install
 
## Run Project

-  npm start
  
## Now you can access on localhost:3000
